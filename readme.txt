An WP Plugin that show/hidde a toolbar with social links of the wp blog/site owner

== Description ==

My Social Links Bar WP is a plugin based on My Social Links Bar JQuery Plugin.

In the bar the WP owner can put her/his social medias links/profiles so people that navigate on site/blog
will be able to access more information and reach the goal of social medias: Connect People, using a simple way for viewing them 
and reduce space inside the site.

In My Social Links Bar the WP owner will be able to: 

* Indicate her/his social medias links (facebook, youtube, twitter, myspace, yahoo, lastfm, vimeo, delicious, flickr, picase, linkedin)
* Set the position of the bar - Top or Bottom / Left or Right


== Installation ==

1. Unzip `my-social-links-bar` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress


== Frequently Asked Questions ==

= What do I need to do after install the plugin ? =

Just put your social media ID in the respective form option. 
The empty social media links won't be visible in the bar.
